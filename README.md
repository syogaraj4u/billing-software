# Billing Software

Browser-based billing software for purchase entry, sales entry, 8 GST profiles, inventory, invoice printing, reports, and optional purchase invoice PDF autofill.

## Features

- Sales entry
- Purchase entry
- 8 GST business profiles
- Item master
- Party master
- Stock updates
- GST reports
- Printable invoice format
- Optional purchase invoice PDF upload/autofill
- PWA support

## GitHub Pages

This app is static HTML, CSS, and JavaScript, so it can run on GitHub Pages.

Publish from the `main` branch root folder.

## Data Storage

Current version stores data in the browser local storage. For shared online data across devices, connect Supabase later.
